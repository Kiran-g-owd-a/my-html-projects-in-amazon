# Crowdsourcing UI Prototypes (MTurk-Style Interfaces)

This repository contains HTML-based microtask interfaces designed to simulate
crowdsourcing workflows often used for data collection, validation, and human-in-the-loop
decision-making. These prototypes demonstrate structured form design, clear instructions,
validation logic, and interactive UI behavior built using HTML, Bootstrap, and JavaScript.

All pages have been recreated using generic content and do not contain confidential information.
